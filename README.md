## Overview
This assignment implements different types of recommender systems using the MovieLens dataset. The following methods are included:

- Content-Based Filtering (TF-IDF, User Profile)
- Collaborative Filtering (User-Based, Item-Based)
- Matrix Factorization (SVD)
- Hybrid Recommendation Model
- Neural Network-based Recommendation
- Reinforcement Learning (Multi-Armed Bandit and Q-Learning)
- Explainability (SHAP, LIME, k-NN)
